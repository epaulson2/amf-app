MI Tension System

Modal interchange chords don't need a new tension set — use parent-key logic instead.
The Parent-Key Rule
What mode am I borrowing from?
→
Find its RELATIVE MAJOR (parent key)
→
Use THAT key's tension sets
Walkthrough — Borrowing from C Minor (Parallel Minor of C)
Borrowing from:
Parallel minor of C = C minor
Parent key:
Relative major of C minor = E♭ major (3 flats)
Therefore:
Use E♭ major's tension sets for all borrowed chords
Example 1:
A♭maj7 (bVImaj7 in C) = IVmaj7 in E♭ → available tensions: 9, #11, 13
Example 2:
Fm7 (iv-7 in C) = ii-7 in E♭ → available tensions: 9, 11
Example 3:
B♭7 (bVII7 in C) = V7 in E♭ → available tensions: 9, 13
